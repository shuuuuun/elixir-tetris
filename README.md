# Elixir Tetris

This is study project.

## Usage

0. Requires [Elixir](https://elixir-lang.org) to be installed.

1. Clone this repository.

2. Install dependencies.
    ```shell
    $ make get
    # equivalent to `mix deps.get`.
    ```

3. Run.
    ```shell
    $ make run
    # equivalent to `MIX_ENV=prod mix run lib/tetris.exs`.
    ```
